# BigInteger in C++
A C++ implementation of unsigned Big Integers, primarily using the source code from
https://www.geeksforgeeks.org/bigint-big-integers-in-c-with-example/
by GeeksForGeeks.

Code is optimized with C++ move scemantics and bug fixes.
